# Vue 3 Test Assignment

Тестовое задание на Vue 3 + Vite + Pinia.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

После сборки откройте **`dist/index.html`** двойным кликом - весь код встроен в один файл (без отдельных `.js` по `file://`, иначе браузер блокирует CORS).

Проверить через локальный сервер (опционально):

```bash
npm run preview
```

## Задания

1. **Интерактивная карточка пользователя** - имя, приветствие, загрузка фото
2. **TODO List** - CRUD задач, статусы, localStorage
3. **Расширенный TODO** - Pinia, счётчики, фильтрация, компоненты
